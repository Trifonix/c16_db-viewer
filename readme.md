# 🚀 test-db — Neon SQLite Browser

Современный браузер баз данных SQLite на **PyQt6** с неоновым интерфейсом, CRUD-операциями и постраничной навигацией.

![PyQt6](https://img.shields.io/badge/PyQt6-6.6+-green)
![SQLite](https://img.shields.io/badge/SQLite-Supported-lightgrey)
![License](https://img.shields.io/badge/License-MIT-orange)

## ✨ Возможности

### 📂 Работа с базами данных

* Открытие файлов `.db`, `.sqlite`, `.sqlite3`
* Автоматическое определение таблиц
* Просмотр количества записей в каждой таблице
* Быстрое переключение между таблицами

### 📋 Просмотр данных

* Отображение данных в удобной таблице
* Автоматическое растягивание столбцов
* Поддержка больших таблиц через пагинацию
* Навигация между страницами данных

### 🛠 CRUD-операции

* ➕ Добавление записей
* ✏ Редактирование записей
* ❌ Удаление записей
* 🔄 Обновление данных без перезапуска приложения

### 🎨 Современный интерфейс

* Neon Cyberpunk UI
* Плавные анимации переходов
* Эффекты прозрачности и затухания
* Адаптивная цветовая схема
* Темная тема по умолчанию

## ⚙️ Установка

### Зависимости

```bash
pip install PyQt6
```

или

```bash
pip install -r requirements.txt
```

---

## 🚀 Запуск

### Обычный запуск

```bash
python test-db.py
```

После запуска выберите SQLite-файл через интерфейс.

---

### Запуск с тестовой базой данных

```bash
python test-db.py --sample
```

Будет автоматически создан файл:

```text
sample.db
```

со следующими таблицами:

* users
* products

---

### Открытие БД через аргумент командной строки

```bash
python test-db.py my_database.db
```

Приложение автоматически загрузит указанную базу данных.

## 🎨 Особенности реализации

### Анимации

Используются:

* QPropertyAnimation
* QGraphicsOpacityEffect
* QEasingCurve

Для создания:

* плавного появления страниц;
* эффектов нажатия кнопок;
* анимированного переключения экранов.

### Архитектура

Основные компоненты:

| Компонент       | Назначение                         |
| --------------- | ---------------------------------- |
| DatabaseBrowser | Главное окно                       |
| FadeStack       | Анимированное переключение страниц |
| TableCard       | Карточка таблицы                   |
| RecordDialog    | Диалог CRUD                        |
| AnimatedButton  | Кнопки с анимацией                 |

---

## 🔐 Ограничения

Для редактирования и удаления записей таблица должна содержать **PRIMARY KEY**.

Если первичный ключ отсутствует, приложение предупредит пользователя о невозможности безопасного изменения данных.
